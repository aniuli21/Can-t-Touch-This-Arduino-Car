# Can-t-Touch-This-Arduino-Car

**Components:**
* Arduion UNO
* Arduino USB Cable
* L293D Motor Driver 
* 3 x Ultrasonic Sensors
* 2 x 9V Batteries
* 1 Breadboard
* 2 x DC motors 
* 1 x Chassis
* 2 x Wheels
* 1 x Ballpin Wheel

**Instructions:**

Connect Arduino UNO to power then start aproaching the car just to see it running from you

**Code is available in:** [proiect_v2.ino](./proiect_v2.ino)
